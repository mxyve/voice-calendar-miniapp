# 📅 语音智能日历 - 前端
一款以语音交互为核心的日历管理工具，支持语音添加/查询/删除日程，集成通义千问大模型与阿里云语音服务。

## 1. 项目简介
本项目为科幻玻璃态风格的微信小程序，基于 uni-app + Vue3 开发，依托通义千问大模型与阿里云语音服务，以语音交互为核心实现日程管理。

**主要特点：**
- 🎙️ 语音交互：支持语音添加、查询、删除日程
- 🤖 AI 智能问答：通义千问解析用户意图
- 🎨 科幻玻璃态：毛玻璃效果、天空渐变背景
- 📱 微信小程序：一键授权登录

---

## 3. 核心功能
| 模块 | 功能说明 |
| --- | --- |
| 用户认证 | 微信一键登录/登出，token 持久化 |
| 日历视图 | 月视图、月份切换、日程圆点标记 |
| 日程管理 | 手动/语音 增、查、删 |
| AI 助手 | 文字/语音对话、意图解析、语音播报 |

**语音交互流程：**
- 用户点击麦克风 → 录音（16000采样率、mp3格式）→ 后端 ASR 识别 → 通义千问意图解析 → 执行日程操作 → TTS 语音播报结果

## 5. 快速开始
### 5.1 项目模板
```
https://gitee.com/dcloud/uni-preset-vue/repository/archive/vite.zip
```

### 5.2 环境要求
- HBuilderX 4.0+ / VS Code
- 微信开发者工具
- Node.js 16+

### 5.3 安装依赖
```bash
npm install
npm i @dcloudio/uni-ui
npm i sass -D
npm install pinia
npm i pinia-plugin-unistorage
```

### 5.4 运行项目
```bash
npm run dev:mp-weixin
```

### 5.5 打包发布
```bash
npm run build:mp-weixin
```

## 6. 页面说明
### 6.1 登录页
- 微信一键授权登录，获取用户头像和昵称
- 科幻玻璃态视觉设计：天空渐变背景、毛玻璃云朵装饰
- 登录成功后自动跳转首页

### 6.2 首页（日历）
- 日历卡片：月视图显示日期，支持上/下月切换
- 日程红点：有日程的日期显示小红点标记
- 日程列表：选中日期后展示该日期的日程
- 左滑删除：支持左滑日程项进行删除
- 手动添加：点击「+ 新增」打开新增弹窗
- 语音入口：右下角麦克风按钮打开 AI 助手

### 6.3 个人中心
- 展示用户头像、昵称
- 退出登录功能（清除 token，返回登录页）
- 毛玻璃卡片风格，与整体主题统一

### 6.4 AI 语音助手弹窗
- 文字输入：支持键盘输入问题，发送后 AI 回复
- 语音输入：按住说话，松开自动识别
- 语音配置：mp3 格式、16000 采样率、单声道、48000 比特率
- 语音播报：AI 回复自动播放语音，支持手动播放/停止
- 对话记录：展示用户与 AI 的对话历史

### 6.5 新增日程弹窗
- 表单字段：标题（必填）、日期（必填）、时间（可选）、备注（可选）
- 日期选择器：支持选择年月日
- 时间选择器：小时 + 分钟（每5分钟一档）
- 毛玻璃背景风格，确定按钮渐变蓝色
